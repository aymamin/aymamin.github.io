Dimensions Master - Privacy Policy

Effective Date: November 2025

1. Overview

The Dimensions Master plugin for AutoCAD is developed by Ayman Amin. This privacy policy explains how user information is handled when using the application.

2. Data Collection

Dimensions Master does not collect, store, or transmit any personal information, drawing content, or user data. All operations are performed entirely within AutoCAD on the user’s local machine.

3. Data Usage

The plugin calculates total lengths of drawing elements such as lines, polylines, arcs, and splines. The results are displayed in a local form window inside AutoCAD. No data is sent or saved externally.

4. Internet Access

Dimensions Master does not require an internet connection and does not connect to any online services, servers, or APIs.

5. Third-Party Sharing

This application does not share any information with Autodesk or any third parties.

6. User Control

Since no personal data is collected, users do not need to take any action to protect or manage data within the plugin.

7. Updates

If the functionality of the plugin changes in the future in a way that affects privacy, this policy will be updated accordingly.
