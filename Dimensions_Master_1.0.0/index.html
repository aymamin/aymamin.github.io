DIMENSIONS MASTER

A powerful yet easy-to-use AutoCAD plugin designed to calculate the total length of specified Entities in your drawing.

Overview

Dimensions Master provides engineers, architects, and designers with quick and accurate length summaries for linear elements such as cables, pipes, boundaries, and utility networks — all displayed clearly in a dedicated results window.

Key Features

  • Instantly calculate the total length of geometry in your AutoCAD drawing.
  • Supports Lines, Polylines, Arcs, and Splines, individually or combined.
  • Choose to include specific layers or all layers.
  • Filter by object color or analyze all colors.
  • Results displayed in a custom form window, with clear values and formatting.
  • Fast, accurate, and user-friendly — ideal for detailed length estimation.
  • Fully compatible with AutoCAD environments used for infrastructure, utilities, and engineering workflows.

How It Works

  1. Open your drawing in AutoCAD.
  2. Run the command DM.
  3. Define which object types to include — lines, polylines, arcs, splines, or any combination.
  4. Optionally select a layer or color filter or leave the default value (--All Layers-- & --All Colors--).
  5. The plugin scans the drawing and displays the total length results in a clear, organized form window, allowing you to review and copy the values easily.

Use Cases

  • Estimating pipe, cable, or duct lengths in infrastructure utility networks.
  • Measuring road centerlines, plot boundaries, or paths in civil and architectural layouts.
