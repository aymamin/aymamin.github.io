Privacy Policy

Effective Date: January 2026

1. Overview

The Coordinates Master plugin for AutoCAD is developed by AYMAMIN. This privacy policy explains how user information is handled and complies with Autodesk Publisher requirements.

2. Data Collection and Usage

Coordinates Master does not collect, store, or transmit personal information or drawing content. All coordinate calculations and actions occur locally within AutoCAD on the user’s machine.

The plugin does not use analytics, advertising, or tracking mechanisms.

The application may perform a secure online check with Autodesk services to verify user entitlement (license status). This process is handled by Autodesk and does not provide the developer with access to personal user data.

3. Third-Party Data Sharing

Coordinates Master does not share any user data with the developer or third parties.

License entitlement verification is performed through Autodesk services. Any data processed during this verification is handled according to Autodesk’s privacy policies.

4. Data Retention and Deletion

No user data is stored or retained by Coordinates Master. All processing happens in memory during runtime and is cleared when AutoCAD closes.

5. User Control

Since no personal data is collected or stored by the plugin, users do not need to manage permissions or data settings. If future versions introduce data-handling features, users will be notified and given appropriate choices.

6. Internet Access

Coordinates Master operates primarily offline.

An internet connection may be required to verify license entitlement through Autodesk services. No user data is transmitted to the developer during this process.

7. Policy Updates

If future versions of Coordinates Master introduce new features involving any data usage or transmission, this policy will be updated accordingly and republished at the same official URL.
