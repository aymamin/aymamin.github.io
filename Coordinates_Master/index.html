COORDINATES MASTER

Overview

Length Master is a powerful yet easy-to-use AutoCAD plugin designed to automatically generate and plot coordinates tables from drawing objects.

It provides engineers, architects, and designers with quick and accurate coordinate data for Blocks, Points, or both, clearly organized in a professional table with leaders linking each object to its corresponding coordinates.

Key Features

  • Instantly creates tables of coordinates with leaders pointing to drawing objects.
  • Supports Blocks and Points, individually or combined.
  • Filter Blocks by block name, layer, color, or any combination.
  • Filter Points by layer, color, or both.
  • Full control over leader geometry, including length, angle, arrow size, and layer.
  • Customize table properties such as table layer, text style, and coordinate labels, with full control over which coordinates (X, Y, Z) are displayed.
  • Fast, accurate, and user-friendly — ideal for precise coordinate documentation.
  • Fully compatible with AutoCAD environments used for surveying, infrastructure, and engineering workflows.

How It Works

  1. Open your drawing in AutoCAD.
  2. Run the command CMR in command line or simply press “Coordinates Master” icon in Add-ins tab.
  3. Choose the object type to include — Blocks, Points, or both.
  4. Apply filters such as block name, layer, or color, or leave default values (--All Layers-- & --All Colors--).
  5. Configure leader geometry and table settings as needed.
  6. The plugin generates a clear, organized coordinates table with leaders pointing to each selected object.
  7. Output screen displays number of blocks and/or points processed successfully.

Use Cases

  • Documenting block and point locations in civil, architectural, and infrastructure projects.
  • Generating accurate coordinate schedules for construction set-out and field execution.
  • Extracting coordinates for utility networks, equipment layouts, and control points.
  • Preparing clear coordinate references for shop drawings, approval drawings, and as-built documentation.
  • Reducing manual errors and saving time when coordinating design data with surveyors and site teams.
