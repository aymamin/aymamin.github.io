Privacy Policy

Effective Date: November 2025

1. Overview

The Length Master plugin for AutoCAD is developed by AYMAMIN. This privacy policy explains how user information is handled and complies with Autodesk’s Publisher requirements.

2. Data Collection and Usage

Length Master does not collect, store, or transmit any personal information or drawing content. All calculations (such as total lengths of lines, polylines, arcs, and splines) occur entirely within AutoCAD on the user’s local machine.

The plugin does not use analytics, ads, or any tracking mechanisms. All operations run locally in memory without accessing or sharing data externally.

3. Third-Party Data Sharing

Length Master does not share any data with Autodesk or third parties. If integrations are added in the future, they will follow the same strict privacy standards.

4. Data Retention and Deletion

No data is stored or retained. All processing happens in memory during runtime and is cleared when AutoCAD closes.

5. User Control

Since no personal data is collected, users don’t need to manage any permissions or settings. If data-handling features are added later, users will be notified and may opt out.

6. Internet Access

Length Master operates fully offline and never connects to the Internet or external servers.

7. Policy Updates

If future versions of Length Master introduce new features involving connectivity or data usage, this policy will be updated accordingly and republished at the same official URL.
